## Features

- Seleccionando una pokebola, aparecerá un pokemon con sus respectivas caracteristicas.

## Tecnologías

- [React Native] 
- [Typescript] 
- [Redux toolkit] 
- [PokéAPI]


## Instalación
Descargar el repositorio
```sh
git clone https://github.com/rube4489/pokemon-app.git

```

## Instalar las dependencias
En el directorio del proyecto, ejecutar
```sh
NPM
npm install

YARN
yarn install

```
## Ejecución
 ```sh
NPM
npm run start

YARN
yarn run start

```
## Android
 ```sh
NPM
npm run android

YARN
yarn run android

```
## IOS
 ```sh
NPM
npm run ios

YARN
yarn run ios

```
